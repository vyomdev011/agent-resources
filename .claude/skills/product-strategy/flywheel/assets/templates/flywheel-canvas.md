## Flywheel Quality Check

> Validate your flywheel against these criteria.

```
┌─────────────────────────────────────────────────────────────────┐
│ QUALITY CHECKLIST                                               │
│                                                                 │
│ [ ] 4-6 components (simple enough to remember)                  │
│                                                                 │
│ [ ] Each component is verb + noun (actionable)                  │
│                                                                 │
│ [ ] Each connection has clear causation (not just correlation)  │
│                                                                 │
│ [ ] The loop closes (last component feeds first)                │
│                                                                 │
│ [ ] It's self-reinforcing (momentum compounds over time)        │
│                                                                 │
│ [ ] It's unique to us (reflects our competitive advantage)      │
│                                                                 │
│ [ ] Team can draw it from memory                                │
│                                                                 │
│ [ ] We have evidence for at least some connections              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
