# Category Skill One

The first nested skill in the _test-nested-skills package for testing nested package structures.

## Usage

This skill is nested within a category directory inside a package, testing that nested skills in packages are properly discovered and flattened.

## Instructions

When this skill is invoked:
1. Confirm you are executing cat-skill-one from the category namespace
2. Report that nested skills within packages are handled correctly
3. Verify the flattened name includes the category path
